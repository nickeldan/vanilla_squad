size_t  size,
const char *  format,
  ... 
)

Signal-safe version of snprintf.

If -1 is not returned, then a null-terminator is guaranteed to be written.

Parameters
bufferThe destination buffer.
sizeThe number of bytes in the buffer. This must be at least 1 so that a null-terminator can be written.
formatThe format string. See the README for the list of supported %-tokens.
Returns
The number of characters (not counting the null-terminator) that were actually written. If either buffer or format is NULL or if size is 0, then -1 is returned.

Same as vasqSafeSnprintf but adjusts the output pointer and remaining capacity.

Where vasqSafeSnprintf takes a char* and a size_t, this function takes pointers to those objects. It prints the buffer in the usual way but then adjusts the char* and size_t so that subsequent calls to this function can pick up where the last one left off.

Parameters
outputA pointer to the location of the next write. This value is adjusted when the function returns.
capacityA pointer to the remaining number of bytes left in the buffer starting at *output. This value is adjusted when the function returns.
formatSame as for vasqSafeSnprintf.
Returns
Same as for vasqSafeSnprintf.
